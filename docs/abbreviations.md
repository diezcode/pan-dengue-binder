# List of Abbreviations

This glossary defines common abbreviations used in this repository. The project uses these terms in the context of safe, computational dengue sequence and structure analysis.

## Core Dengue Terms

| Abbreviation | Meaning | Short explanation |
|---|---|---|
| ADE | Antibody-Dependent Enhancement | A phenomenon where antibodies bind a virus without fully neutralizing it and may help the virus enter Fc-receptor-bearing immune cells. In dengue, ADE is one reason safety review is especially important. |
| DENV | Dengue virus | The virus that causes dengue disease. |
| DENV-1 | Dengue virus serotype 1 | One of the four main dengue serotypes. |
| DENV-2 | Dengue virus serotype 2 | One of the four main dengue serotypes. |
| DENV-3 | Dengue virus serotype 3 | One of the four main dengue serotypes. |
| DENV-4 | Dengue virus serotype 4 | One of the four main dengue serotypes. |
| E protein | Envelope protein | A major dengue surface protein often studied for binding, structure, and immune recognition. |
| EDIII / DIII | Envelope Domain III | A domain of the dengue E protein. It is often discussed as one possible target region in computational analyses. |
| EDE | E-dimer epitope | A quaternary epitope formed by the dengue E-protein dimer. It is important in literature on broadly neutralizing antibody recognition. |
| NS1 | Nonstructural protein 1 | A dengue protein involved in disease biology and immune recognition, but not the same as a direct virion surface target. |
| Serotype | Viral subtype group | Dengue has four main serotypes: DENV-1, DENV-2, DENV-3, and DENV-4. |

## Immunology and Safety Terms

| Abbreviation | Meaning | Short explanation |
|---|---|---|
| Ab | Antibody | An immune protein that binds a target antigen. |
| Ag | Antigen | A molecule or molecular feature recognized by antibodies or immune receptors. |
| BnAb | Broadly neutralizing antibody | An antibody that can neutralize multiple related viral types or strains. |
| Fc | Fragment crystallizable region | The antibody region that can interact with immune-cell receptors. Fc activity is relevant to ADE risk discussions. |
| FcR | Fc receptor | A receptor on some immune cells that binds antibody Fc regions. |
| Fc-silent | Fc receptor-silenced | A modified antibody-like format intended to reduce or remove Fc receptor interaction. |
| IgG | Immunoglobulin G | A common antibody class in blood. |
| Neutralization | Blocking infection-related activity | In this repo, this should be treated as a hypothesis or assay endpoint, not as a proven claim. |

## Sequence and Structure Analysis

| Abbreviation | Meaning | Short explanation |
|---|---|---|
| AA | Amino acid | A building block of proteins. |
| FASTA | FASTA sequence format | A common text format for biological sequences. |
| MSA | Multiple sequence alignment | An alignment of many related sequences used to compare conservation and variation. |
| PDB | Protein Data Bank / PDB file format | A database and file format for biomolecular structures. |
| mmCIF | Macromolecular Crystallographic Information File | A modern structure file format used by the PDB. |
| RCSB | Research Collaboratory for Structural Bioinformatics | Organization that operates the RCSB Protein Data Bank. |
| SASA | Solvent-accessible surface area | A structural measure estimating how exposed a residue or atom is to solvent. |
| RMSD | Root-mean-square deviation | A structural difference metric between two conformations. |
| UniProt | Universal Protein Resource | A major protein sequence and annotation database. |
| NCBI | National Center for Biotechnology Information | A major biological data resource. |

## Computational Design and Prediction

| Abbreviation | Meaning | Short explanation |
|---|---|---|
| AF2 | AlphaFold2 | A protein structure prediction system. |
| ColabFold | ColabFold | A practical AlphaFold-based prediction workflow. |
| RFdiffusion | RoseTTAFold Diffusion | A generative protein structure design method. |
| ProteinMPNN | Protein Message Passing Neural Network | A model commonly used to design amino acid sequences for protein backbones. |
| pLDDT | Predicted Local Distance Difference Test | A confidence metric commonly reported by AlphaFold-style models. |
| PAE | Predicted Aligned Error | A metric describing predicted positional uncertainty between residues. |
| ipTM | Interface predicted Template Modeling score | A confidence metric often used for predicted protein complexes. |

## Experimental Terms Mentioned for Planning Only

These terms may appear in roadmap documents as non-actionable validation concepts. This repository should not provide wet-lab protocols.

| Abbreviation | Meaning | Short explanation |
|---|---|---|
| BLI | Bio-layer interferometry | A method used by qualified labs to measure binding interactions. |
| SPR | Surface plasmon resonance | A method used by qualified labs to measure binding affinity and kinetics. |
| VLP | Virus-like particle | A non-replicating particle-like system used in some research contexts. |
| PK | Pharmacokinetics | How a drug or biologic moves through the body over time. |
| CMC | Chemistry, Manufacturing, and Controls | Regulatory and manufacturing documentation for medical products. |
